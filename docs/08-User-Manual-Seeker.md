# User Manual — Job Seeker

## Getting Started
1. Register at `/register.html` as "Job Seeker"
2. Log in at `/login.html`
3. You'll land on the Seeker Dashboard

## Dashboard
- **Stats cards:** Applications, Average Score, Shortlisted, Pending
- **Recent Applications:** Quick view of latest applications with status
- **ATS Score Trend:** Line chart of your scores across applications
- **Quick Actions:** Upload Resume, Browse Jobs, Recommendations

## Uploading Resumes
1. Click "Upload Resume" from sidebar or dashboard
2. Drag-drop or browse for PDF/DOCX file (max 5MB)
3. System extracts text and runs NLP analysis
4. View all your resumes in "My Resumes"

## Resume Quality Check
- Click "Quality Check" on any resume
- Get standalone formatting/structure analysis without applying to a job
- See detected sections, missing sections, contact info present

## Resume Version Comparison
- Select two resumes in "Compare Resumes"
- See side-by-side: word count, skills, sections
- Identify what changed between versions

## Browsing & Applying to Jobs
1. Go to "Browse Jobs"
2. Filter by: Search, Job Type, Location, Experience
3. Click any job → view details
4. "Apply Now" → select resume → submit
5. System runs async ATS analysis against that specific job

## Tracking Applications
- "My Applications" shows all in pipeline
- Status badges: Applied → Under Review → Shortlisted → Interview → Offer → Hired
- Pipeline visualization shows progress
- Click "View Report" for full ATS analysis

## Analysis Report
- **Score Doughnut:** Overall 0-100 with grade (A+ through F)
- **6 Score Categories:** Progress bars for each
- **Matched/Missing Skills:** Green/red chip badges
- **Matched/Missing Keywords:** With confidence percentages
- **Experience Analysis:** Action verbs detected, weak phrases found
- **Achievements:** Measurable accomplishments extracted
- **Improvement Suggestions:** Prioritized by impact with estimated score gain
- **What-If Simulator:** Check skills you plan to add → see projected score

## AI Recommendations
- Based on your latest resume skills
- Shows top 5 matching jobs with match percentage
- Displays matched and missing skills per job
- "Apply Now" takes you directly to job detail

## Notifications
- Bell icon in top bar shows unread count
- Notifications for: application received, status changes, interview, score updates
- Mark as read / mark all read
